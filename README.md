# haemoglobinopathy-variant
# Evaluation of in silico tools for variant effect prediction of haemoglobinopathies
## Project overview
The aim of this study is to evaluate and compare the performances of selected in silico tools for variant prediction in the classification missense variants in HBA1,HBA2 and HBB genes and to assess their contributions to evidence-based variant interpretation within the ACMP/AMG framework,especially in the PP3/BP4 evidence
## Objectives
The objectives of this project is to
1.Curate a high quality dataset of variants in the HBA1,HBA2 and HBB genes 
2.Collect and compile prediction scores fron selected tools including AlphaMissense and EVE.
3. Evaluate the performance of each prediction tool using metrics such as  specificity,sensitivity, accuracy and overall concordance 
4.Compare the agreement and disagreement between diffrent prediction tools and identify the factors that influence their influence
5.With focus on the PP3/BP4 criteria of the ACMG/AMP framework, assess the contribution of prediction tools variant classification
