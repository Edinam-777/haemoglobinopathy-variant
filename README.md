# haemoglobinopathy-variant
# Evaluation of in silico tools for variant effect prediction of haemoglobinopathies
## Project overview
The aim of this study is to evaluate and compare the performances of selected in silico tools for variant prediction in the classification missense variants in HBA1,HBA2 and HBB genes and to assess their contributions to evidence-based variant interpretation within the ACMP/AMG framework.
## Objectives
Curate a high quality dataset of missense variants in the HBA1,HBA2 and HBB genes 
Collect and compile prediction scores fron selected tools including AlphaMissense and EVE.
Evaluate the performance of each prediction tool using metrics such as specificity,sensitivity, accuracy and overall concordance 
Compare the agreement and disagreement between diffrent prediction tools and identify the factors that influence their influence
With focus on the PP3/BP4 criteria of the ACMG/AMP framework, assess the contribution of prediction tools variant classification
